# Public Prize Contest Radar Brief

목표
- 공모전 포털 전체를 뒤지는 것이 아니라, 사용자 강점과 수상 가능성이 높은 공공형 상금 공모전의 발생 지점과 반복 패턴을 추적한다.
- 종합 포털, 공식 채널, seed 기관을 병행 감시하되 결과물은 “지원 가치 높은 후보”만 남긴 digest 로 축약한다.

핵심 가설
1. 공공기관·지자체 정책/아이디어 제안형
2. 시민참여혁신단·국민평가단·모니터링단
3. 공공홍보 숏폼/영상/UCC
4. 공공데이터·AI·창업/경진대회형
5. 서포터즈/플레이어/선정형 활동

운영 원칙
- 1~4번이 본진, 5번은 보조 트랙
- 현재 모집 공고보다 종료 이력/수상작 페이지를 더 중요한 지도 데이터로 취급
- 원문 발생지와 포털 재게시를 분리 저장
- 저효율 유형은 초기에 감점 또는 제외

1차 산출물
- 과거 공모전 DB 스키마
- 공모전 유입 경로 지도
- 사용자 적합 분야 우선순위표
- 매일/매주 감시할 핵심 소스 리스트
- BrowserOS/CDP 기반 동적 목록/상세 수집 루틴
- Telegram digest 및 마감 임박 알림 전송 루틴
